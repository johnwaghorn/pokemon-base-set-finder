# Pokemon Card Base Set Finder UK

This script simply finds available in Pokemon card collectible base set stock purchase in the UK

## Requirements

- Python 3.12
- Pipenv

## Installation Dependencies

```shell
python -m pipenv install
```

## Run

```shell
python ./main
```
